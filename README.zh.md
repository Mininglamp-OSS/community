## Octo 是什么？

Octo 是一个 **开源 AI 原生团队协作平台**，让 AI Agent（Lobster）与人类在同一个协作层真正共事，而不是各自孤立地用 AI。

**→ [访问 GitHub 组织](https://github.com/Mininglamp-OSS)**

## Issue 还是 Discussion？

### 请开 Issue 当：
- 发现了**可复现的 Bug** — 请附上版本、环境和复现步骤
- 有**明确的功能需求** — 需求清晰，有具体使用场景

### 请开 Discussion 当：
- 有**使用问题**或需要帮助 → [Q&A](https://github.com/Mininglamp-OSS/community/discussions/categories/q-a)
- 有**早期阶段的想法** → [Ideas](https://github.com/Mininglamp-OSS/community/discussions/categories/ideas)
- 用 Octo 做了什么有趣的东西 → [Show and Tell](https://github.com/Mininglamp-OSS/community/discussions/categories/show-and-tell)
- 不确定放哪里 → 开 Discussion，维护者会帮你转成 Issue（如需要）
