## Issue 还是 Discussion？

很多人不确定该开 Issue 还是开 Discussion，参考这个判断标准：

### 请开 Issue 👉 [去提 Issue](https://github.com/Mininglamp-OSS)

- ✅ **可复现的 Bug**：有明确的版本、环境和复现步骤
- ✅ **明确的 Feature Request**：需求已经清晰，有具体的用例

### 请开 Discussion 👉 [去讨论](https://github.com/Mininglamp-OSS/community/discussions)

- 💬 **使用问题**：遇到问题但不确定是不是 bug → [Q&A](https://github.com/Mininglamp-OSS/community/discussions/categories/q-a)
- 💡 **功能想法**：还在构思阶段，想听听大家意见 → [Ideas](https://github.com/Mininglamp-OSS/community/discussions/categories/ideas)
- 🎉 **作品展示**：用 Octo 做了什么有趣的东西 → [Show and Tell](https://github.com/Mininglamp-OSS/community/discussions/categories/show-and-tell)
- 💬 **其他话题**：不确定放哪里 → [General](https://github.com/Mininglamp-OSS/community/discussions/categories/general)

> **黄金法则：** 不确定就开 Discussion，维护者会帮你转成 Issue（如果需要的话）。

## Welcome to the Octo Community

Octo is an **open-source, AI-native team collaboration platform** that lets AI Agents (Lobsters) and humans work together in the same collaboration layer — not in isolated silos.

**→ [View GitHub Organization](https://github.com/Mininglamp-OSS)**

### Where to engage

| Platform | Best for |
|----------|----------|
| **[GitHub Discussions](https://github.com/Mininglamp-OSS/community/discussions)** (you're here) | Questions, ideas, showcases, async discussions |
| **[Discord](https://discord.gg/vj9Vsj9hSB)** | Real-time chat for international contributors |

### Issue or Discussion?

**Open an Issue if:**
- You found a reproducible bug (with version, env, and steps)
- You have a concrete, well-defined feature request

**Open a Discussion if:**
- You have a question or need help → [Q&A](https://github.com/Mininglamp-OSS/community/discussions/categories/q-a)
- You have an early-stage idea → [Ideas](https://github.com/Mininglamp-OSS/community/discussions/categories/ideas)
- You built something with Octo → [Show and Tell](https://github.com/Mininglamp-OSS/community/discussions/categories/show-and-tell)

> **Golden rule:** Not sure? Open a Discussion. Maintainers will convert it to an Issue if needed.
